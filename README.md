# proton_tools
Tools for managing proton mailbox

## Remove mail dupplicate
```
python remove_dupplicates.py
```

## Sync all .elm file of a specific folder into proton INBOX
```
python sync_elm.py
```

## Mark all mail unread as read
```
python mark_all_read.py
```

You need to install proton bridge on your machine and rename the .env.sample to .env and fill the required information.
